# TalentSift — AI Recruitment Intelligence

An AI-powered recruitment tool that automatically ranks job candidates by ATS compatibility score.

## Live Demo
🔗 https://talentsift-production.up.railway.app

## What it does
- Upload 50+ resumes at once
- Paste a job description
- AI analyzes and ranks every candidate
- Shows ATS score, matching skills, missing skills, and hiring recommendation

## Tech Stack
- Python + Flask
- Groq AI (Llama 3.3)
- Cloudinary (cloud storage)
- Railway (deployment)
- HTML + CSS
